CERITA TERBENTUK NYA OMA MART

Waroeng oma Awalnya terbentuknya waroeng oma pada 2 juni tahun 2023, dengan modal barang untuk dijual kembali senilai 3.000.000 (tigajuta), Modal barang warung itu dibiayakan dari orangtua saya tepatnya ibu, dengan modal segitu memang menurut saya sikit tapi saya yakin bahwa warung ini bisa berkembang. Barang dagangan yg saya jual kembali itu tidak la banyak tapi melainkan bervariasi jadi saya harus mencari toko grosir untuk membeli barang lalu dijual lagi, dan allhamdulillah kurang dari 1 tahun saya bisa menjadikan untung penjualan menjadi modal untuk memperbanyak barang dagangan dan untung penjualan untuk kebutuhan saya sendiri. Nama waroeng oma itu saya dapat karna memang rata-rata ditanjung pinang orang nyebut warung itu “kedai” dalam otak saya jgn kedai warung aja dan saya pun mengubah dengan kata-kata yang warung menjadi waroeng seperti orang zaman dulu menulis setau saya sih gitu, kalau oma itu karna ponakan saya manggil ibu saya itu “oma” , jadi saya buat aja “waroeng oma” letak waroeng oma itu di Jalan Batu Naga perum Pesona Alam Mutiara II blok B6. Untuk kepemilikan waroeng oma sendiri saya tetap pada pendirian saya yaitu ibu saya. Kenapa waroeng oma bisa berdiri awalnya itu karna saya pengganguran lulus sma. Kurang lebih 1 tahun saya kerja ,jualan makanan,bengkel motor yg punya abg sodara, kurir paket, dan ojek online. Kurang dari satu tahun warung ini berdiri banyak suka dan duka, suka karna saya bisa ngasih uang ke ibu saya, bisa beli barang yg saya inginkan. Kalau duka itu paling berkesan bagi saya ketipu 1 juta di bulan ke 3 dan lucunya ibu saya ga marah malah ketawa ibu saya bilang “tu la kau kena tipu kan” memang ibu saya ga minta ganti uang satu juta itu tapi saya berusaha untuk ganti, allamdulillah dapat saya ganti uang itu, pembelian barang biasa saya beli sendiri ditoko grosir biar murah yg pasti, saya ambil untung itu pakai persen dari jenis barang dan harga barang paling besar saya ambil untung 15% dari harga barang dan jenis barangnya rata-rata biasa sih di 5-10% contoh rokok modal 30.000 tambah 10% jadi 33.000
